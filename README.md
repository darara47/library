# Library - back-end

Application to manage library.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Database

```bash
# generate migration
$ npm run migrations:generate

# run migration
$ npm run migrations:run

# run seeds
$ npm run seed
```
